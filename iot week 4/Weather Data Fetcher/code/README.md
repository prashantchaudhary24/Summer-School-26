# Air Quality Monitoring System

## Problem Statement

Monitor indoor air quality using MQ2 and DHT11.

## Hardware

- Arduino UNO
- MQ2 Gas Sensor
- DHT11
- RGB LED
- Buzzer
- HC05 Bluetooth Module

## Features

- Gas Detection
- Temperature Monitoring
- Humidity Monitoring
- RGB Status
- Buzzer Alarm
- Bluetooth Alert
- CSV Logging
